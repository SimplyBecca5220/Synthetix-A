## Overview

**Syn-A** is an AI-driven "Yield-as-a-Service" agent designed to solve liquidity fragmentation within the Mantle 3.0 ecosystem. By leveraging Mantle’s modular architecture, Syn-A automatically identifies and captures the highest risk-adjusted yields for **$mETH** and **$cmETH** (Restaked ETH) across native protocols like Agni Finance, Merchant Moe, and Pendle.

## The Problem

In the current Mantle landscape, yield is everywhere, but it is **fragmented**. 
- **Cognitive Load:** Manually tracking APR spikes across multiple DEXs is a full-time job.
- **Inactivity:** Most users "stake and forget," missing out on secondary rewards like **$COOK** or optimized restaking spreads.
- **Gas Inefficiency:** On L1, frequent rebalancing is too expensive. On Mantle, it’s a competitive advantage.

## The Solution: How Syn-A Works

Syn-A acts as an autonomous brain for your capital:
1. **The Scout:** A specialized logic model (SynAgent.ts) monitors Mantle subgraphs for yield anomalies.
2. **The Strategist:** It calculates the most efficient route between LSTs and LRTs (mETH ↔️ cmETH).
3. **The Executor:** Transactions are routed through a secure **SynVault** on Mantle, executing "Micro-Rebalancing" to capture gains as small as 0.5%—which is only viable thanks to Mantle's low fees.

## 🛠️ Mantle Onchain Integration

### 1. Liquid Staking Flywheel ($mETH & $cmETH)
Syn-A is built to be "Mantle-Native." It prioritizes assets that contribute to Mantle’s TVL, specifically focusing on the utility of **Mantle Staked Ether** and **Mantle Restaked Ether**.

### 2. Modular Data Availability (DA)
To ensure transparency without high costs, Syn-A utilizes **Mantle DA**. Every rebalancing "Intent" generated by the AI is logged on-chain, allowing users to audit exactly why the agent moved funds.

### 3. Secure SynVault (Solidity)
The core execution layer is a Solidity-based vault that features:
- **Whitelisted Logic:** The AI agent can only interact with pre-approved Mantle protocols.
- **Non-Custodial Design:** Users retain ultimate "Owner" rights to withdraw funds at any time.
